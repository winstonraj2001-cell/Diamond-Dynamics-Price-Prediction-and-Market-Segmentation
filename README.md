# 💎 Diamond Analytics Dashboard

An end-to-end **Machine Learning Web Application** that predicts diamond prices and segments diamonds into market categories using clustering.

---

## 🚀 Live Demo

🔗 https://diamond-dynamics-price-prediction-and-market-segmentation-cp99.streamlit.app/

---

## 📌 Features

* 🔮 Price Prediction (Random Forest)
* 📊 Market Segmentation (KMeans)
* 📈 Interactive Dashboard
* 🔍 Dynamic Filters
* 🔐 Login System
* 🤖 AI Chatbot (basic)
* 💾 SQLite Database
* 📥 CSV Download

---

## 🧠 ML Models

* Random Forest Regressor (~92% accuracy)
* KMeans Clustering (3 segments)

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas, NumPy
* Scikit-learn
* Plotly
* SQLite
* Google Drive (model hosting)

---

## ⚙️ Run Locally
